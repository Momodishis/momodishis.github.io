Tebra Primary Care Website Redesign

Overview


The aim of redesigning the doctor's website's main platform was to create a more user-friendly and intuitive experience that would attract new clients and improve the experience of existing clients. To achieve this goal, the first step involved identifying any existing usability issues within the current platform.


This was done through user testing and feedback from both existing clients and the target audience. Based on the collected information, critical features were identified for prioritization. This project represents a significant advancement towards the ultimate goal of enhancing user experience of the website.
